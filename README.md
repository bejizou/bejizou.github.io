# Portfolio – ELbeji Meherzia

## 📌 Présentation

Ce dépôt contient le code source de mon portfolio professionnel en ligne.  
Il présente mon parcours, mes compétences et mes projets en tant qu’Ingénieure Test Logiciel.

---

## 🛠️ Technologies utilisées

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  
- Git / GitHub  
- Outil d’audit d’accessibilité : WAVE  

---

## 🚀 Fonctionnalités

### 🔹 Section À propos
Présentation de mon profil, de mon parcours et de mes objectifs professionnels.

### 🔹 Section Compétences (dynamique)
- Générée automatiquement via le fichier `skills.json`
- Affichage dynamique des cartes de compétences

### 🔹 Section Projets (dynamique)
- Générée automatiquement via le fichier `portfolio.json`
- Chaque projet contient :
  - Une image
  - Une description
  - Un lien vers le code source

### 🔹 Section Contact
- Formulaire de contact
- Message de confirmation après envoi
